Hides the Update Reminder in the Admin for all non Admin users.

== Description ==

This plugin allows you to hide the update WordPress reminder from a specific user levels. By default it hides from anyone who cannot update core files.

If you have multiple users then this means you can hide the notice from them. Useful for CMS based sites, so the client doesn't see the notice.

== Installation ==

Installation Instructions:

1. Download the plugin and unzip it.
2. Put the 'hide-update-reminder' folder into your wp-content/plugins/ directory.
3. Go to the Plugins page in your WordPress Administration area and activate the Hide Update Reminder.
